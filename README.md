# FastAPI Auth - Frontend

Frontend em React para a API de autenticação e gestão de tarefas.

## 🔗 Links
- **Live Demo:** https://fast-api-auth-react.vercel.app
- **Backend API:** https://fastapi-auth-rest-production.up.railway.app

## 🛠️ Stack
- React
- JavaScript
- CSS

## ✨ Funcionalidades
- Registo e login de utilizadores
- Autenticação com JWT
- Gestão de tarefas (criar, editar, apagar, filtrar)

## 🚀 Correr localmente

```bash
git clone https://github.com/RubenSilva13/FastApi-Auth-React
cd FastApi-Auth-React
npm install
npm start
```

## 📡 Configuração
O frontend comunica com a API em:
```
https://fastapi-auth-rest-production.up.railway.app
```
